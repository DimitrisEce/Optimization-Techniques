# Optimization-Techniques
Matlab projects regarding the optimization of given functions.

The Notebook file is the first and simplest part of the assignment. It is about optimizing 2-D function. The second part, shown in Iterative Methods 1 optimizes a given 3-D function through various iterative methods (Newton's, Levenberg-Marquardt, Steepest Descent). The last part explores the Steepest Descent with Projection method with different parameters for initialization and step size.

For more details inspect the Reports for each part.
